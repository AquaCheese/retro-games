## API Endpoints

### GET /api/catalog
Returns list of available games with install status
```json
{
  "games": [
    {
      "id": "road-blocks-2023",
      "title": "Road Blocks",
      "installed": false,  â† Shows if it's on your Pi
      ...
    }
  ]

```

### POST /api/install/:gameId
Downloads and installs a game to your Pi
```bash
curl -X POST http://localhost:3000/api/install/road-blocks-2023
```

### DELETE /api/games/:gameId
Removes an installed game from your Pi
```bash
curl -X DELETE http://localhost:3000/api/games/road-blocks-2023
```

### GET /api/games
Lists all currently installed games
```bash
curl http://localhost:3000/api/games
```
